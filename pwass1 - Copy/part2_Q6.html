HTML TAGS

  1. html: Represents the root element of an HTML document.
  2. head: Contains meta-information about the HTML document, such as title, links to stylesheets, scripts, etc.
  3. body: Encloses the visible content of the HTML document, including text, images, videos, and other elements.
  4. h1 to h6: Headings that define different levels of importance (from highest to lowest).
  5. p: Represents a paragraph of text.
  6. a: Defines a hyperlink, which allows you to link to another webpage or resource.
  7. img: Embeds an image into the HTML document.
  8. ul: Creates an unordered list, usually represented by bullet points.
  9. ol: Represents an ordered list, where items are numbered.
  10. li: Defines a list item within either an ul or ol list.
