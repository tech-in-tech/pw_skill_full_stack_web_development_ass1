HTML/CSS/JavaScript

The fundamental building blocks of web development:

  • HTML: Defines the structure and content of web pages.
  • CSS: Handles the presentation and styling of web pages.
  • JavaScript: Enables interactive and dynamic elements on web pages.
Front-end Frameworks

Frameworks that simplify and enhance front-end development:

  • React: A popular library for building component-based user interfaces.
  • Angular: A full-fledged framework for creating dynamic web applications.
  • Vue.js: A progressive framework for building user interfaces.
Back-end Frameworks

Frameworks used for server-side development and handling business logic:

  • Node.js: A JavaScript runtime that allows running JavaScript on the server side.
  • Express: A minimal and flexible Node.js framework for building web applications.
  • Django: A high-level Python web framework that encourages rapid development.
  • Ruby on Rails: A web application framework based on Ruby programming language.
Database Management Systems (DBMS)

Software used for storing, managing, and retrieving data:

  • MySQL: An open-source relational database management system.
  • PostgreSQL: A powerful, open-source object-relational DBMS.
  • MongoDB: A NoSQL database using a document-oriented data model.
Version Control

Tools to manage changes and collaboration in the development process:

  • Git: A distributed version control system to track changes in code.
  • GitHub/GitLab: Web-based platforms for hosting Git repositories and collaboration.
Deployment Platforms

Services to deploy and host web applications:

  • Heroku: A cloud platform supporting multiple programming languages.
  • Netlify: A platform for continuous deployment and hosting static sites.
  • AWS/Azure/GCP: Cloud service providers offering various deployment options.
